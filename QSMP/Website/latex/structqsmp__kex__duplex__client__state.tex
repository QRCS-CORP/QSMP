\doxysubsection{Detailed Description}
Internal state for the Duplex key exchange (client-\/side). 

This structure holds the state information required by a client participating in a Duplex key exchange. It includes\+:
\begin{DoxyItemize}
\item {\ttfamily keyid\+:} A unique key identity string (of size {\ttfamily QSMP\+\_\+\+KEYID\+\_\+\+SIZE}) that identifies the key exchange session.
\item {\ttfamily schash\+:} A session token hash (of size {\ttfamily QSMP\+\_\+\+DUPLEX\+\_\+\+SCHASH\+\_\+\+SIZE}) used to verify session integrity.
\item {\ttfamily prikey\+:} The client\textquotesingle{}s asymmetric cipher private key (of size {\ttfamily QSMP\+\_\+\+ASYMMETRIC\+\_\+\+PRIVATE\+\_\+\+KEY\+\_\+\+SIZE}).
\item {\ttfamily pubkey\+:} The client\textquotesingle{}s asymmetric cipher public key (of size {\ttfamily QSMP\+\_\+\+ASYMMETRIC\+\_\+\+PUBLIC\+\_\+\+KEY\+\_\+\+SIZE}).
\item {\ttfamily rverkey\+:} The remote party\textquotesingle{}s asymmetric signature verification key (of size {\ttfamily QSMP\+\_\+\+ASYMMETRIC\+\_\+\+VERIFY\+\_\+\+KEY\+\_\+\+SIZE}).
\item {\ttfamily sigkey\+:} The client\textquotesingle{}s asymmetric signature signing key (of size {\ttfamily QSMP\+\_\+\+ASYMMETRIC\+\_\+\+SIGNING\+\_\+\+KEY\+\_\+\+SIZE}).
\item {\ttfamily ssec\+:} The derived asymmetric shared secret (of size {\ttfamily QSMP\+\_\+\+SECRET\+\_\+\+SIZE}) computed during key exchange.
\item {\ttfamily verkey\+:} The client\textquotesingle{}s local asymmetric signature verification key (of size {\ttfamily QSMP\+\_\+\+ASYMMETRIC\+\_\+\+VERIFY\+\_\+\+KEY\+\_\+\+SIZE}).
\item {\ttfamily expiration\+:} A timestamp (in seconds from the epoch) indicating when the key exchange session expires. 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
The server connection collection. 

This header file defines internal functions for managing the collection of connection state objects used by the QSMP server. The connections collection maintains an array of connection states (of type {\ttfamily \doxylink{structqsmp__connection__state}{qsmp\+\_\+connection\+\_\+state}}), each representing an active or available socket connection. The functions declared herein allow for\+:


\begin{DoxyItemize}
\item Checking the active status of a connection slot.
\item Adding new connection states to the collection.
\item Retrieving connection states by instance or index.
\item Initializing and disposing of the connections collection.
\item Resetting and clearing connection states.
\item Determining the capacity and availability of the collection.
\item Running self-\/tests to ensure the collection operates correctly.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
These functions are internal and non-\/exportable. 
\end{DoxyNote}

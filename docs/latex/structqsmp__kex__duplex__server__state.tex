\doxysubsection{Detailed Description}
Internal state for the Duplex key exchange (server-\/side). 

This structure holds the state information required by a server participating in a Duplex key exchange. It contains cryptographic parameters including key identities, session hashes, asymmetric keys, and an expiration timestamp. In addition, it includes a callback function ({\ttfamily key\+\_\+query}) that is used to retrieve the appropriate public key during the key exchange process.
